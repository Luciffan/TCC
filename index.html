Introdução ao Controle PID

Neste projeto, transformamos a teoria complexa da engenharia de controle em uma aplicação prática. O objetivo foi desenvolver um robô de duas rodas capaz de se equilibrar autonomamente utilizando o algoritmo PID.

O conceito central é o sistema de malha fechada, onde o controlador observa o estado do robô e corrige constantemente sua posição para evitar a queda.

Justificativa e Mudança de Rota

Inicialmente projetado como um pêndulo invertido tradicional (haste sobre carrinho), o projeto migrou para o Robô Autobalanceado de Duas Rodas (TWIP) após falhas estruturais no primeiro protótipo. O princípio físico permanece o mesmo, mas a aplicação robótica torna o projeto mais próximo de tecnologias comerciais reais, como os Segways.

Objetivos do Projeto

Com o projeto redefinido, nosso objetivo principal é: Implementar e validar um sistema de controle Proporcional para a estabilização de um protótipo de robô autobalanceado, adquirindo conhecimento prático sobre seus componentes e algoritmo de controle.

Para isso, definimos dois objetivos secundários:

  • Discutir a relevância e os desafios da aplicação do controle PID em sistemas de mobilidade, utilizando o robô como um estudo de caso prático.
  • Construir um protótipo funcional de baixo custo, utilizando o microcontrolador Arduino, o sensor MPU-6050 e atuadores adequados.
